
<h1>supermall</h1>
<h3>1.项目简介</h3>
<ul>
  <li>该项目为使用vue-cli4搭建的一个前后端分离的商城项目；</li>
  <li>使用了Vue[包括混入mixin，过滤器filters相关等]，Vuex，Vue-Router,ES6，Axios以及Webpack等前端技术；</li>
  <li>主要包含商城首页，详情页，购物车，商品分类，个人中心等模块；</li>
  <li>项目涉及vue-lazyload, better-scroll相关运用。</li>
</ul>
